```bash

# Start with PM2
cd backend
pm2 start ecosystem.config.js
```

#### Frontend (Vercel)
```bash

# Option 2: Deploy via GitHub (Recommended)

# 3. Deploy automatically on push to main

npm run build
vercel --prod --local-config ../vercel.json
```
